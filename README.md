# Assignment 2 – Route Academy

This project is the **second assignment in my Frontend Development journey at Route Academy**.

The goal of this assignment is to practice building page layouts using **basic HTML and CSS techniques only**, without relying on modern layout systems.

## 📌 Project Goal

In this assignment, the layout was created using **traditional CSS methods** instead of modern tools.

The main objective was to understand how layouts were built before Flexbox and Grid, and to strengthen the fundamentals of CSS positioning and structure.

## 🧱 Layout Rules

For this project, the layout was implemented using:

* **Float**
* **Inline-block**

The following modern layout techniques were **not used**:

* ❌ Flexbox
* ❌ CSS Grid

Also, the styling was kept **simple**, focusing only on basic CSS properties to practice core concepts.

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**

## 📚 What I Practiced

Through this assignment, I practiced:

* Structuring a webpage with **clean HTML**
* Creating layouts using **float and inline-block**
* Working with **basic CSS properties**
* Organizing page sections properly
* Building forms and UI elements
